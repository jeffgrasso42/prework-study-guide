HTML

  • The body element represents the visible content shown to the user.

CSS

  • margin
    • A margin property indicates how much space we want around the outside of an element.
    • The margin is not visible on the page; it merely adds a buffer.
  • padding
    • Often confused with margin, but it is not the same thing.
    • The padding property adds space around the content inside an element.
    • When you increase padding, you are actually making the entire element bigger!
  • border
    • The border shorthand CSS property sets an element's border.
    • It sets the values of border-width, border-style, and border-color
  • display
    • sets whether an element is treated as a block or inline element and the layout used for its children
